Density of States (DOS) Visualisation

An intuitive and interactive exploration of how dimensionality affects the density of states of an ideal Fermi gas.

Python Statistical Mechanics

Overview

This project investigates how the density of states (DOS) depends on dimensionality by combining analytical derivations with numerical visualisation. The goal is to build intuition around phase space arguments rather than treating the DOS as a memorised formula.

Implementation

The visualisation is implemented in Python using NumPy and Matplotlib. Sliders allow interactive comparison across 1D, 2D, and 3D cases.
